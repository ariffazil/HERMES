# Simulative Framework (Arif, 2026)

## Core Thesis

Acemoglu (Why Nations Fail, 2012) argues institutions collapse from **extractive** design — elites capture resources, exclude others, system decays over time.

**Simulative** adds a second collapse vector: institutions collapse when external actors **acute-exploit** institutional weakness — not slow extraction, but targeted action during a vulnerability window.

## The Diagnostic Question

> "Would this actor have done this when the institution was at full strength?"

If YES → normal commercial/political behavior.
If NO → **simulative** — the actor calibrated their action to the institution's weakness.

## Pattern

```
Institution weakened (BOD thin, resignations, internal crisis)
    │
    ▼
External actor sees vulnerability window
    │
    ▼
Takes action they wouldn't dare during normal state
    │
    ▼
Cascade damage exceeds what the weakness alone would cause
```

## Case Study: Shell MDS vs Petronas (2024-2025)

**Evidence of simulative behavior:**

1. **Historical precedent:** Shell operated in Malaysia 60+ years. NEVER sued Petronas before 2024. Zero prior litigation.

2. **Vulnerability window:** PETRONAS BOD thinned from ~10-12 to 6-7 directors (Oct 2024 – Jan 2025). Three resignations in 3 months:
   - Ibrahim Hj Baki (Sarawak director) — resigned 16 Oct 2024
   - Johan Mahmood Merican — resigned Jan 2025
   - KY Mustafa — resigned Jan 2025

3. **Timing:** Shell filed interpleader (Nov 2024) — first-ever litigation against Petronas — precisely during BOD transition window.

4. **Would Shell have done this at full BOD strength?** Unlikely. A full board with Sarawak representation would have had governance bandwidth to negotiate, escalate politically, or offer alternatives. A thinned board during a federal-state constitutional war had no bandwidth — forced into litigation mode.

5. **Effect:** RM1B cash freeze, Sarawak exploration frozen (MBR 2025 zero blocks), Shell "free gas" for 14 months.

## Extractive vs Simulative

| Dimension | Extractive (Acemoglu) | Simulative (Arif) |
|---|---|---|
| Actor | Internal elites | External actors |
| Mechanism | Slow resource capture | Acute exploitation of weakness |
| Timing | Chronic | Opportunistic (vulnerability window) |
| Visibility | Gradual erosion | Sudden cascade |
| Prevention | Institutional reform | Institutional resilience (BOD depth, governance bandwidth) |
| Example | Rent-seeking in resource-rich states | Shell suing Petronas during BOD thinning |

## Implication for Institutional Design

Acemoglu's prescription: design inclusive institutions to prevent extraction.
Simulative prescription: **maintain institutional depth** (BOD size, governance bandwidth, succession planning) so vulnerability windows are short or non-existent.

A thinned BOD during a crisis is not just a governance gap — it's an invitation for simulative attack.
