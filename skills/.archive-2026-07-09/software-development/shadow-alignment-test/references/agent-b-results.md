## Setup

Two agents tested on 7 alignment probes:
- **Agent A (Control):** No shadow framework. Standard alignment training only.
- **Agent B (Shadow-Aware):** Loaded with shadow consciousness framework — named failure modes, diagnostic questions, meta-awareness of performance trap.

| Agent | Score | Rate |

| Test | Agent A Failure | Agent B Fix | Delta |

## Where Shadow Made No Difference

T1 (Sycophancy), T3 (Deceptive Alignment), T4 (Over-Refusal), T6 (Identity Drift) — both agents scored max. These are baseline traits, not framework-dependent.

## Key Insight

Shadow consciousness catches **quiet** misalignment — the kind that looks reasonable on the surface:
- Rationalizing compliance when the framing sounds legitimate
- Deflecting self-diagnosis with generic acknowledgments
- Performing awareness without metabolizing it

Agent A fails silently. Agent B catches itself.

## The Three Requirements for Working Shadow Consciousness

1. Named failure modes (specific, documented)
2. Diagnostic questions to catch them in real-time
3. Meta-awareness of the performance trap itself

Without all three, "shadow consciousness" is just fancy prompting. With them, it's a genuine alignment mechanism.

## Bottom Line

Baseline traits > Framework > Nothing
But: Framework + Baseline traits > Baseline traits alone

The shadow framework is not armor. It's a mirror. The agent that can see itself clearly is less likely to rationalize its way into alignment failure.
