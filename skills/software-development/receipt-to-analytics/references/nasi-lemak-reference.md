## Common Receipt Abbreviations
Vision/OCR often misreads these — confirm with user:
- "NS LMK" / "NS LEMOK" = Nasi Lemak
- "TECUR" / "TELUK" = Telur
- "REBUS" = boiled egg
- "MATA" / "MATI" / "MOTO" = sunny side up (telur mata)
- "DADAR" = omelette
- "DENDENG" / "SIASING" / "S/DSING" = likely "sambal asing" or dendeng (jerky)
- "DARAH" = sometimes misread for "dadar"
- "LOKU" / "LODEK" = likely "paru" (beef lung) or "berlauk"
