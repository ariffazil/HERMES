# Group Chat Trading Signal Discussions

## Context

When trading signals come up in casual group chats (not formal trading channels), the AI should provide technical observations with proper disclaimers. User validated this approach: "ni betul aku aku pakai camni".

## Response Pattern

When asked for a trading signal (e.g., "bagi signal sell"):

### 1. Current Price & Bias
- State current price from chart
- Confirm trend direction (bullish/bearish)
- Note key price action (spike, rejection, consolidation)

### 2. Entry Zones
- Buy zone: specific price range
- Sell zone: specific price range
- Use "HERE" or "REJECT" labels for clarity

### 3. Risk Management
- Stop loss level with dollar/pip risk
- Take profit targets (T1, T2)
- Risk:Reward ratio

### 4. Confirmation Signals
- What candlestick pattern to wait for
- What price level confirms the setup
- What invalidates the setup

### 5. F2 Disclaimer
Always include: "ini observation only, bukan financial advice. Always manage risk!"

## Example Response

```
XAUUSD H1 — Gold dropped hard from 4137 → 4065 area! 📉

**Bearish bias confirmed:**
- High spike rejected at 4137
- Lower highs pattern forming

**Current setup:**
- Price at 4065 — mid-range
- Sell on re-test of 4084-4090 zone (rejection)
- Break below 4050 → target 4006 support

⚠️ F2: observation only, bukan financial advice!
```

## Key Insight

The user trades XAUUSD (Gold) using:
- H4 = macro trend direction, H1/M15 = entry timing
- Key support/resistance levels
- Candlestick confirmation patterns
- Risk:Reward ratios (aim for 1:3+)
- CPI/news awareness for entries
- Multiple concurrent positions (e.g., 3x SELL 0.3 at different entries)
