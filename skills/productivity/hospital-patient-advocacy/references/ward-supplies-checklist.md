# Ward Supplies Checklist for HKL

## Critical Items (Buy Immediately)

| Item | Detail | Price |
|---|---|---|
| Pampers dewasa (tape type) | Patient bedridden. Tape type easier for nurses to change than pants type. | Elderjoy RM 14-18/pack (10 pcs), MamyPoko RM 25-35, Depends RM 20-30 |
| Baju button depan | Front-opening for easy IV drip access. Avoid pull-over shirts. | ~RM 15-25 each |
| Seluar longgar elastic waist | No pressure on surgical wound | ~RM 20-30 |

## Comfort Items

- Tuala kecil x2
- Stokin (hospital air-cond sejuk)
- Bantal extra (tinggikan kepala untuk GERD/comfort)
- Tisu basah + tisu kering
- Flask air suam
- Lip balm (bibir kering lepas NPO/tube)
- Sikat

## Bila Dah Boleh Jalan

- Slipper untuk ke tandas

## Where to Buy Near HKL

| Store | Location | Best For |
|---|---|---|
| Guardian | Inside HKL, ground floor | Most convenient |
| Watson | Opposite HKL, Jalan Pahang | Good selection |
| KK Mart / 99 Speedmart | Along Jalan Pahang | Cheapest pampers |

## Priority Order

1. Pampers (tanpa ni, susah untuk nurse)
2. Baju button depan (senang tukar dengan IV drip)
3. Tuala kecil + tisu
4. Everything else
