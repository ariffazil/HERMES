## Kaparinyo

**Origin:** Minangkabau (West Sumatra). Popularized by Gumarang (1960s), Elly Kasim (1972), revived by Siti Nurhaliza. The word "kaparinyo" likely derives from Portuguese "cafrinho" via colonial-era cultural exchange.

### Traditional Version (Gumarang / Elly Kasim)

```
[Intro]
Ondeh sayang... kaparinyo...

[Verse 1]
Kaparinyo, babuai adiak babuai
Kaparinyo, babuai adiak babuai
Ondeh sayang
Babuai, babuai piciangkan mato
Yo malang, yo malang
Babuai piciangkan mato

[Verse 2]
Kaparinyo, kok lah dapek kawan sasuai
Kaparinyo, kok lah dapek kawan sasuai
Ondeh sayang
Jan lah lupo, jan lah lupo jo kawan lamo
Yo malang, yo malang
Jan lah lupo jo kawan lamo

[Verse 3]
Kaparinyo, duo tigo kuciang balari
Kaparinyo, duo tigo kuciang balari
Ondeh sayang
Ndak sarupo, ndak sarupo jo kuciang balang
Yo malang, yo malang
Ndak sarupo jo kuciang balang

[Verse 4]
Kaparinyo, duo tigo buliah dicari
Kaparinyo, duo tigo buliah dicari
Ondeh sayang
Ndak sarupo, ndak sarupo jo nan hilang
Yo malang, yo malang
Ndak sarupo jo nan hilang
```

### Siti Nurhaliza Version Pantun (bridge/interlude)

```
Orang berinai berhitam kuku
Mandi dijirus si air mawar
Jikalau sampai hasrat hatiku
Racun kuminum jadi penawar

Sungguhlah harum si bunga tanjung
Hati nak petik si bunga mawar
Rindu dan dendam tidak tertanggung
Budi setitik jadi penawar
```

### Oslan Husein Version (extended)

```
Kaparinyo, kaparinyo lah lagu usali
Kaparinyo, lagu lah usali
Marilah dendangkan kok samo, basamo lah sama
Mari manari basamo samo (oi kanduang)
Mari manari basamo samo

Kaparinyo, kirai salendang lah rantakkan kaki
Kaparinyo, rantakkan lah kaki
Tuo lah jo mudo kok tari, marilah manari
Tuo jo mudo mari manari (manari)
Tuo jo mudo mari manari
```

### Key Minang Vocabulary

| Minang | Meaning |
|---|---|
| babuai | farewell / goodbye |
| adiak | younger sibling / dear |
| ondeh sayang | oh dear love (exclamation) |
| yo malang | oh fate / oh woe |
| piciangkan mato | avert your gaze |
| kok lah dapek | if you find |
| kawan sasuai | suitable companion |
| jan lah lupo | don't forget |
| kawan lamo | old friends |
| duo tigo | two or three |
| kuciang balari | cats running |
| ndak sarupo | not the same |
| nan hilang | what is lost |
| usali | ancestral / traditional |
| rantakkan kaki | stamp the feet |
| tuo jo mudo | old and young |
| oi kanduang | oh dear ones |
