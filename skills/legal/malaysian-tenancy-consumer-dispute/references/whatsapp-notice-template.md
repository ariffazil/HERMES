# WhatsApp Notice Template (Bahasa Malaysia)

## For Landlord / Contractor Breach

```
Assalamualaikum [NAMA]. Ni notice TERAKHIR.

Saya [NAMA PENUH].
[ISI SPESIFIK: tenant unit X, kontrak start X]
[Dah bayar penuh / bayaran dibuat]

Tapi sampai sekarang:

1. [ISU PERTAMA]
2. [ISU KEDUA]
3. [ISU KETIGA]

Saya dah [X] hari/minggu [duduk tanpa aircond / tunggu servis / dll].

SAYA BAGI TEMPOH SAMPAI [TARIKH SPESIFIK] SAHAJA.

2 pilihan:
1. [SETTLE SEMUA DI ATAS] SEGERA
2. REFUND PENUH [duit sewa / bayaran / deposit]

Kalau [TARIKH] masih gagal, SAYA [TERMINATE KONTRAK / BUAT TUNTUTAN].
Saya [tuntut melalui TTPM / lapor pihak berkuasa].
Takkan runding lagi.

Ini notice terakhir.
```

## Tactical Notes

- **Always include a specific date** — not "hujung minggu ni" but "27/7/2026"
- **Always give exactly 2 options** — settle OR refund. No third path.
- **Always state consequence** — TTPM, terminate, report. Real, not empty threat.
- **"Ini notice terakhir"** — signals this is the final communication before action.
- **Tone**: No "harap maklum", no "kerjasama tuan", no softeners. Direct. Assertive.

## For Tenant Stay-Strategy Follow-up (post-deadline)

```
[NAMA LANDLORD],

Tempoh saya bagi dah tamat [TARIKH].
Tuan gagal selesaikan.

Saya dengan ini MENAMATKAN kontrak serta-merta.

Saya tuntut refund:
- [SENARAI AMOUNT]

Sila transfer dalam 7 hari.
Kalau gagal, saya fail TTPM tanpa notis lanjut.

Sekian.
```
