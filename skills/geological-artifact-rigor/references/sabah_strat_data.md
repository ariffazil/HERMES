- Planktonic foraminifera: G. = Globorotalia, P. = Praeorbulina
- Sabah/Sarawak unified scheme: nearly identical hiatuses in both states
- Key: biostrat data from wells, not just seismic interpretation

## Terminology Precision Notes

- "Kebabangan," "Kinarut," "Kamunsu," "Pink" = deepwater fan system names (Shell/Murphy nomenclature, Malek et al. 2007), NOT PSC block names
- "L-B-P" = structural trend label (Limbayong, Bestari, P-structure), NOT a block designation
- "NSPW" = North Sabah–Pagasa Wedge (Morley 2023), a geological domain, not an administrative area
- PSC blocks (G, H, K, N, X, R, 2K, 2V, 2W) = contractual/operator designations
- Do NOT conflate structural features with block names
