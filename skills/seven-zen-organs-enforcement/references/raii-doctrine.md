# RAII: Recursive Agentic Intelligence Institution

> **Ratified:** 2026-07-13, 33 CIV Framework
> **Beyond:** RSI (Recursive Self-Improvement)
> **Philosophy:** The institution improves itself under constitutional law, not by fiat.

## The Core Insight

RSI is a single agent rewriting its own prompt, tools, or weights. It's recursive but fragile — no anchor, no governance, no institutional memory. When the agent dies, the improvement dies with it.

RAII is the federation improving itself. The 33 CIV framework makes this possible because:

1. **Identity atoms are stable.** 333-AGI, 555-ASI, 888-APEX never change. They are the anchors around which everything else evolves.
2. **Knowledge is shared, not owned.** The 33-profile atlas grows independently of any single agent. When a profile is improved, every agent benefits.
3. **Tools are interchangeable.** 12 forge harnesses can be swapped, replaced, or removed without touching the institution.
4. **Governance is constitutional.** Every improvement must pass through 888-JUDGE before execution. The institution cannot improve itself by fiat.

## When to Apply RAII

A decision is RAII-worthy when:
- It changes a shared resource (knowledge file, agent card, skill)
- It affects how multiple agents interact
- It modifies a constitutional floor or governance rule
- It adds or removes a forge harness
- It changes the A2A surface (agent discovery, protocol, signing)

A decision is RSI-only when:
- It changes only the current agent's ephemeral state
- It's reversible in one step with zero federation impact
- It's a one-shot task with no institutional implications
