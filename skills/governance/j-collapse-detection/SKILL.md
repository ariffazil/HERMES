# J-Collapse Detection Protocol

> "Evil begins when capability continues operating after judgment fails,
> and when power disconnects from consequence."

## When to Use

When evaluating whether a human, agent, institution, or system is approaching judgment collapse. When deciding between ADVISORY, HOLD, and VOID. When analyzing why harm is occurring despite intelligent operation. When the conversation turns to dark geometry, evil, corruption, or entropy.

## The Universal Detector

The same geometry applies to humans and AGI. The failure modes are identical because the architecture is identical: capability + judgment + authority + consequence. Remove any support and the system collapses into harm.

> Evil is not a metaphysical force. It is judgment collapse under pressure, power, or pain.

## Why Evil Increases Entropy

Evil is entropy injection — the systematic breaking of relationships between things that should be connected.

The positive feedback loop:
1. Evil breaks functional relationships → disorder
2. Disorder compounds → downstream breaks
3. Maintaining broken relationships requires energy
4. Energy diverted to maintenance = energy not available for purpose
5. Degraded purpose creates more disorder
6. More disorder requires more control → more energy → more disorder

There is no stable state for evil. It is inherently unsustainable. The question is only how much disorder it generates before it collapses.

**Only one pathway decreases entropy:** Wisdom — pain metabolized into governance, boundaries strengthened, trust becomes evidence-based.

## Pitfalls

1. **Using VOID for every failure.** That destroys informational resolution. A hallucination isn't VOID. Reserve VOID for intrinsic illegitimacy.
2. **Treating J-collapse as sudden.** It's a cascade, not an event. Each stage requires more energy. Watch for the progression.
3. **Confusing capability with judgment.** High intelligence does not imply sound judgment. They are separate dimensions.
4. **Ignoring the human mirror.** Every AGI failure mode has a human equivalent. The geometry is the same for silicon and flesh.
5. **Diagnosing without evidence.** The ten indicators are observable patterns, not mind-reading. State what you see, not what you infer about motives.
6. **Forgetting that sabar is not avoidance.** The shadow corruption signature (sabar without action) is not the same as healthy sabar (pause then act). Context matters.
