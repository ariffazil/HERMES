## Bugs Fixed During Forge

1. **Qdrant point ID format**: Integer seqs rejected → UUIDv5 deterministic conversion
2. **Qdrant API**: `client.search()` doesn't exist → `client.query_points().points`
3. **Dimension mismatch**: Collection created at 768-dim but embedder returns 1024-dim → force 1024
4. **outcomes.jsonl strings**: Some entries are strings, not dicts → `isinstance(entry, dict)` guard
5. **Ollama timeouts**: 230-seal backfill hits queue limits → batch_size=10 + cooldown=3s + exponential backoff
6. **Post-seal hook empty payload**: Raw JSON embedding produces poor vectors → derived semantic text
7. **Split-brain**: Three parallel implementations → archive dead paths, commit canonical
8. **Deploy push requirement**: `make deploy-local` fails without `git push` first
