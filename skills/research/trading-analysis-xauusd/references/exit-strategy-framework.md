# Exit Strategy Framework — XAUUSD

## When to Sell

Selling is harder than buying — emotions get in the way. Use rule-based exits, not feelings.

### Technical Exit Signals
- **TP hit** — take it. TP is TP. Don't chase "lagi sikit naik."
- **Candle rejection at resistance** — bearish engulfing, shooting star at key level
- **Structure break** — if the reason you entered is no longer valid, exit
- **EMA cross** — EMA 20 crosses below EMA 50 (bearish signal)
- **RSI overbought (>70)** + bearish candle confirmation

### Rule-Based Exit Framework
| Rule | Action |
|------|--------|
| Pre-set TP | Decide BEFORE entry, not after |
| Trailing stop | Lock profit as price moves in your favor |
| Time-based | If trade doesn't move in 24-48hrs, reassess |
| Structure break | If support/resistance that justified entry breaks, exit |

### The One Rule
> Exit when the **reason you entered** is no longer valid. Not when you feel like it.

### Emotional Traps
| Trap | Fix |
|------|-----|
| "Lagi sikit naik" | TP is TP. Take it. |
| Takut rugi so cut awal | Trust your R:R setup |
| Revenge trade after loss | Walk away, come back tomorrow |
| Dah profit tapi tak sell | Greed = account killer |

### Key Levels for Exit Decisions (example: entry 4022.93)
| Level | Action |
|-------|--------|
| $4,100-4,120 | Resistance pertama — consider partial TP |
| $4,200 | Resistance kuat — good TP zone |
| $4,000 | Support — kalau break, cut loss |
| $3,950 | Stop loss area |

### R:R Verification
- Always calculate risk vs reward before entry
- Minimum 1:2 R:R to justify the trade
- Partial close: close portion at TP1, runner to TP2
