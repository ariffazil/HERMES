# Reunion Chat Variant — Nostalgia + Attendance Ambivalence

When a user sends a reunion group chat (school batch, college alumni, workplace reunion), the dynamics are distinct from event planning or behavioral profiling.

## Structural Markers
- Group created recently (weeks/months before event)
- Old photos shared in bulk — "aib gila", "muka sendiri pun xkenal", laughing at teenage selves
- Identity confusion — people can't match faces to names, forget homeroom/class names
- Committee formation — AJK roles, polls for dates/locations, repeated roster reminders
- Nostalgia bonding — "aib" (embarrassing) photo sharing is an affectionate ritual, not humiliation

## Emotional Context: The User Is In The Chat
Unlike analyzing someone else's conversation, the user is a PARTICIPANT. They:
- See everything you see
- Have personal history with these people
- Are processing their OWN feelings about attending

When the user expresses reluctance ("I kinda feel not going", "So many scars"):
1. **Name what the chat reveals** — the warmth, the genuine desire to reconnect
2. **Name what you know about their current state** — career struggles, introversion, life weight
3. **Do NOT push attendance** — the ambivalence is real and valid
4. **Offer one honest observation** about what they'd be walking into
5. **End with space** — "the question isn't really should I go, it's am I in a state where I can show up as I am"

## Low-Signal Presence Mode
During or after emotional processing, the user may shift to low-signal:
- Image without narration → acknowledge presence, don't analyze
- Location pin → brief area recognition, one question max
- Short messages → match their brevity
- This is the user processing through artifacts, not requesting information

## What NOT To Do
- Don't turn it into therapy
- Don't pathologize the reluctance
- Don't dump data tables — the user wants insight, not analytics
- Don't compare attendance numbers to pressure them
- Don't over-interpret — the connection between the chat and their feelings is usually obvious
